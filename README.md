# 📊 Superstore Sales Profit Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes sales, profit, and order performance using the Superstore dataset.

The project provides insights into revenue generation, profitability, product performance, and regional sales trends to support business decision-making.

---

## 🎯 Objectives

- Analyze total sales and profit performance.
- Identify top-performing products and categories.
- Compare regional sales trends.
- Track order quantity and profitability.
- Discover areas with high sales but low profit.
- Support data-driven business decisions.
